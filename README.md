# Dataminr

## Requirements

* Docker: https://www.docker.com/get-started
* Package Manager:
    * Ubuntu: Already has one
    * Mac: https://brew.sh/
* Make
    * Ubuntu: `sudo apt-get install make`
    * Mac: `brew install make`

## Installation

* `make install`

## Running

* `make dev`

## Linting

* `make lint`
* `make lint-fix`
